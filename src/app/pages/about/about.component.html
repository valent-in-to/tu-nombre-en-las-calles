Proyecto integrador final

Esta aplicación fue creada en el marco del proyecto final del curso de Desarrollador Web Frontend de la Asociación Icaro, por mí, Valentin Alvarez.

Que APIs utiliza?

La aplicación consume datos de 3 APIs:

  • API del Servicio de Normalización de Datos Geográficos de Argentina
  • Mapbox API
  • Una api propia para obtener las coordenadas de las ciudades (Link a github)
